# googleShoppingTracker
googleShoppingTracker tracks target products in Google Shopping and save csv file if price drops below target price.
<br>
**- Dependencies -** <br>
selenium version == 3.141.0 <br>
webdriver-manager == 3.2.2 <br>
<br>
Chrome version == 83
